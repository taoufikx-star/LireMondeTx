## 🎯 Description

LireMonde est une bibliothèque en ligne développée pour **ReadSphere**, une startup spécialisée dans la culture numérique.  
Le projet permet d'afficher, rechercher, filtrer et gérer une collection de livres via une API REST simulée avec JSON Server.

---

## ✨ Fonctionnalités

- 📖 Affichage dynamique des livres en grille
- 🔍 Recherche en temps réel par titre ou auteur
- 🏷️ Filtrage par genre (Classique, Science-fiction, Drame, Fantasy)
- 🪟 Modale de détails pour chaque livre
- ❤️ Gestion de la liste "À lire" (ajout / suppression)
- 🛠️ Tableau de bord Admin (ajouter, modifier, supprimer)
- 🌙 Design Dark Mode Vintage
